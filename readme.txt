== Description ==

Customizable banner that features an incredibly simple and intuitive interface


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/simple-promo-banner` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Navigate to settings -> Simple Promo Banner
